## Chmod

### Los roles

- u (Propietario / User): El creador o dueño del archivo.
- g (Grupo): Otros usuarios que pertenecen al mismo grupo que el propietario.
- o (Otros): El resto de usuarios del sistema.
- a (Todos / All): Engloba a los tres anteriores a la vez.

### Los permisos

- r (Lectura / Read): Permite ver el contenido del archivo.
- w (Escritura / Write): Permite modificar el archivo.
- x (Ejecución / eXecute): Permite ejecutar el archivo si es un programa.
